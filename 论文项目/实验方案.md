# 基于 UR10 与 FT300 融合数据采集的铲泥作业参数优化实验方案

## 1. 实验目的

本实验以 UR10 机械臂为执行平台，在末端安装铲具与 FT300 六维力/力矩传感器，通过手动示教和参数化轨迹控制完成泥浆铲取作业。实验结合 WSL2 环境下的数据采集系统，同步采集 UR10 机械臂运动状态数据和 FT300 末端受力数据，用于分析铲泥参数对取料质量、末端受力、能量消耗和运动平稳性的影响。

实验主要目标如下：

1. 建立 UR10 机械臂末端铲具铲泥作业实验流程；
2. 分析下探深度、切入角度、水平切入速度和垂直提起速度对铲泥效果的影响；
3. 利用 FT300 采集铲泥过程中的三向力和三向力矩；
4. 利用 UR10 实时数据获取关节角、关节速度、关节电流、TCP 位姿和 TCP 速度；
5. 建立以取料质量、受力峰值、能量消耗和轨迹平滑性为核心的多目标评价指标；
6. 筛选较优铲泥参数组合，为后续基于视觉—力觉融合的自主铲泥控制提供实验基础。

# 3. 实验变量设计

你原方案中已经确定了四个控制变量：下探深度 (d)、切入角度 (\\theta)、水平切入速度 (v\_c)、垂直提起速度 (v\_l)。其中切入角度需要统一定义，建议定义为：0° 表示铲面水平，90° 表示铲面垂直。

## 3.1 控制变量


| 符号      | 名称         | 定义                                 | 推荐范围       |
| --------- | ------------ | ------------------------------------ | -------------- |
| (d)       | 下探深度     | 铲尖从接触泥浆面到停止下探的垂直距离 | 10～30 mm      |
| (\\theta) | 切入角度     | 铲面与泥浆水平面的夹角               | 15°～45°     |
| (v\_c)    | 水平切入速度 | 铲具沿水平方向切入泥浆的速度         | 0.03～0.08 m/s |
| (v\_l)    | 垂直提起速度 | 铲具从泥浆中向上提起的速度           | 0.03～0.08 m/s |

---

## 3.2 变量定义说明

### 1. 下探深度 (d)

下探深度指铲具接触泥浆表面后，继续向下运动的垂直距离。该参数直接影响铲具进入泥浆的深度，从而影响取料质量和末端阻力。

### 2. 切入角度 (\\theta)

切入角度定义为铲面与水平泥浆表面的夹角。

```text
θ = 0°：铲面水平，类似平铲；
θ = 90°：铲面垂直，类似刀刃向下。
```

实验中不建议直接使用过大的切入角度，否则铲具容易出现卡滞或受力突变。

### 3. 水平切入速度 (v\_c)

水平切入速度表示铲具沿水平方向进入泥浆的速度。速度过低可能导致取料效率低，速度过高可能导致冲击力增大。

### 4. 垂直提起速度 (v\_l)

垂直提起速度表示铲具完成下探后向上提起的速度。该参数会影响泥浆附着量、掉泥情况和提升阶段受力。

# 4. 铲泥动作阶段划分

为了便于示教、数据标记和后续分析，将一次完整铲泥过程划分为五个阶段。


| 阶段 | 名称          | 说明                               | 推荐运动方式     |
| ---- | ------------- | ---------------------------------- | ---------------- |
| S0   | 准备阶段      | 机械臂从安全位运动到泥浆上方准备点 | `movej`          |
| S1   | 接触阶段      | 铲具靠近泥浆表面，铲尖接近接触点   | `movel`          |
| S2   | 切入阶段      | 铲具以设定角度和速度水平切入泥浆   | `movel`          |
| S3   | 下探阶段      | 铲具从泥浆表面下探至设定深度       | `movel`          |
| S4   | 提起/切出阶段 | 铲具带泥从泥浆中提起并离开         | `movel`或`movep` |
| S5   | 回收阶段      | 机械臂回到称重或安全位置           | `movej`          |

对应示教点建议设置为：


| 点位 | 含义               |
| ---- | ------------------ |
| P0   | 安全初始点         |
| P1   | 泥浆表面上方准备点 |
| P2   | 切入起点           |
| P3   | 切入终点           |
| P4   | 下探终点           |
| P5   | 提起终点           |
| P6   | 回收称重点         |

轨迹结构为：

```text
P0 → P1 → P2 → P3 → P4 → P5 → P6
```

# 6. 响应变量与评价指标

原始方案中只设置了单次取料质量 (m) 作为响应变量，即一次铲取后铲具上附着的泥浆质量。

结合现在的 UR10 + FT300 采集系统，建议将响应变量扩展为以下几类。

---

## 6.1 取料质量指标

### 单次取料质量 (m)

[
m = m\_{\\text{after}} - m\_{\\text{before}}
]

其中：

* (m\_{\\text{before}})：空铲具或容器初始质量；
* (m\_{\\text{after}})：铲取后质量；
* (m)：单次铲取泥浆质量。

优化方向：

```text
m 越大越好。
```

---

## 6.2 末端受力指标

### 合力

[
F(t)=\\sqrt{F\_x^2(t)+F\_y^2(t)+F\_z^2(t)}
]

### 最大合力

[
F\_{\\max}=\\max F(t)
]

### 平均合力

[
F\_{\\text{avg}}=\\frac{1}{n}\\sum\_{k=1}^{n}F(k)
]

### 合力标准差

[
F\_{\\text{std}}=\\sqrt{\\frac{1}{n-1}\\sum\_{k=1}^{n}\\left(F(k)-F\_{\\text{avg}}\\right)^2}
]

其中：

* (F\_{\\max})：反映最大受力冲击；
* (F\_{\\text{avg}})：反映平均负载；
* (F\_{\\text{std}})：反映受力波动程度。

优化方向：

```text
Fmax、Favg、Fstd 越小越好。
```

---

## 6.3 末端力矩指标

### 合力矩

[
M(t)=\\sqrt{M\_x^2(t)+M\_y^2(t)+M\_z^2(t)}
]

### 最大合力矩

[
M\_{\\max}=\\max M(t)
]

### 平均合力矩

[
M\_{\\text{avg}}=\\frac{1}{n}\\sum\_{k=1}^{n}M(k)
]

优化方向：

```text
Mmax、Mavg 越小越好。
```

---

## 6.4 机械臂电流负载指标

UR10 采集系统已经能够记录 `Act_I0`～`Act_I5` 六个关节电流。可以利用关节电流构造负载指标。

### 关节电流综合指标

[
I\_{\\text{sum}}(t)=\\sum\_{i=0}^{5}|I\_i(t)|
]

### 平均电流负载

[
I\_{\\text{avg}}=\\frac{1}{n}\\sum\_{k=1}^{n}I\_{\\text{sum}}(k)
]

### 最大电流负载

[
I\_{\\max}=\\max I\_{\\text{sum}}(t)
]

优化方向：

```text
Iavg、Imax 越小越好。
```

这个指标对你很有价值，因为你现在的数据系统已经可以实时显示和保存关节电流。

---

## 6.5 能量消耗估计

如果暂时没有关节力矩数据，可以用 FT300 末端力和 TCP 位移估算作业能量。

[
E\_F=\\sum\_{k=2}^{n}F(k)\\Delta s(k)
]

其中：

[
\\Delta s(k)=\\sqrt{
\\left(X\_k-X\_{k-1}\\right)^2+
\\left(Y\_k-Y\_{k-1}\\right)^2+
\\left(Z\_k-Z\_{k-1}\\right)^2
}
]

这里的 (X,Y,Z) 来自 UR10 的 `Act_X`、`Act_Y`、`Act_Z`，(F(k)) 来自 FT300 的合力。

优化方向：

```text
EF 越小越好。
```

---

## 6.6 轨迹平滑性指标

### 力变化率指标

[
J\_F=\\sum\_{k=2}^{n}\\frac{|F(k)-F(k-1)|}{\\Delta t}
]

### 力突变次数

设定阈值 (\\Delta F\_{th})，例如 10 N 或根据预实验确定。当满足：

[
|F(k)-F(k-1)|>\\Delta F\_{th}
]

记为一次力突变。

最终得到：

[
N\_s=\\sum \\mathbf{1}\\left(|F(k)-F(k-1)|>\\Delta F\_{th}\\right)
]

优化方向：

```text
JF、Ns 越小越好。
```

# 7. 实验设计方案

你的原方案推荐使用响应曲面法，并采用中心复合设计 CCD，4 个因素，每个因素 5 个水平，总计 30 组实验，每组重复 3 次，共 90 次铲取操作。

这个方案适合正式优化，但直接执行 90 次实验工作量较大。因此建议采用“两阶段实验设计”。

---

## 7.1 第一阶段：预实验与安全范围确定

目的：

1. 验证 UR10 轨迹是否安全；
2. 验证 FT300 和 UR10 数据是否能同步保存；
3. 确定参数安全范围；
4. 初步判断哪些参数影响最大。

推荐先做 3 组预实验：


| 组别 | 下探深度 (d) | 切入角度 (\\theta) | 水平切入速度 (v\_c) | 垂直提起速度 (v\_l) | 目的                               |
| ---- | -----------: | -----------------: | ------------------: | ------------------: | ---------------------------------- |
| T1   |        10 mm |               15° |            0.03 m/s |            0.03 m/s | 最保守参数，验证能否正常铲入       |
| T2   |        20 mm |               30° |            0.05 m/s |            0.05 m/s | 中等参数，观察正常铲泥效果         |
| T3   |        30 mm |               45° |            0.08 m/s |            0.08 m/s | 高负载参数，验证是否卡滞或受力过大 |

如果 T3 出现卡滞、报警或 (F\_{\\max}) 明显过大，则正式实验中应降低最大深度或最大速度。

---

## 7.2 第二阶段：正交初筛实验

如果时间有限，建议先将 (v\_l) 固定为 0.05 m/s，只研究：

```text
下探深度 d
切入角度 θ
水平切入速度 vc
```

采用 L9 三因素三水平正交实验。


| 水平 | (d) / mm | (\\theta) / ° | (v\_c) / m/s |
| ---- | -------: | -------------: | -----------: |
| 1    |       10 |             15 |         0.03 |
| 2    |       20 |             30 |         0.05 |
| 3    |       30 |             45 |         0.08 |

L9 实验表如下：


| 试验编号 | (d) | (\\theta) | (v\_c) | 固定 (v\_l) |
| -------- | --: | --------: | -----: | ----------: |
| 1        |  10 |        15 |   0.03 |        0.05 |
| 2        |  10 |        30 |   0.05 |        0.05 |
| 3        |  10 |        45 |   0.08 |        0.05 |
| 4        |  20 |        15 |   0.05 |        0.05 |
| 5        |  20 |        30 |   0.08 |        0.05 |
| 6        |  20 |        45 |   0.03 |        0.05 |
| 7        |  30 |        15 |   0.08 |        0.05 |
| 8        |  30 |        30 |   0.03 |        0.05 |
| 9        |  30 |        45 |   0.05 |        0.05 |

每组至少重复 2～3 次。若今晚时间不足，可以先每组做 1 次，后续对较优组重复。

---

## 7.3 第三阶段：响应曲面优化实验

在正交初筛后，如果确认四个变量都对结果有明显影响，再采用 CCD 响应曲面设计。

正式 CCD 因素为：


| 因素   | 名称                |
| ------ | ------------------- |
| (x\_1) | 下探深度 (d)        |
| (x\_2) | 切入角度 (\\theta)  |
| (x\_3) | 水平切入速度 (v\_c) |
| (x\_4) | 垂直提起速度 (v\_l) |

响应变量为：

```text
m、Fmax、Mmax、EF、JF、Ns、Iavg
```

建立二次回归模型：

[
Y=\\beta\_0+\\sum\_{i=1}^{4}\\beta\_i x\_i+\\sum\_{i=1}^{4}\\beta\_{ii}x\_i^2+\\sum\_{i<j}\\beta\_{ij}x\_ix\_j+\\varepsilon
]

其中 (Y) 可以分别代表取料质量、最大合力、能量消耗或综合评分。

## 8.3 单次实验步骤

每组实验流程如下：

1. 设置本组参数：(d,\\theta,v\_c,v\_l)；
2. 在示教器或程序中确认 P0～P6 路点；
3. 在 GUI 中点击“开始采集”；
4. 系统自动创建数据文件夹；
5. 机械臂运行到 P0；
6. 按下 GUI 标记按钮或 Space 键，将 `Label` 设置为 1；
7. 执行铲泥轨迹；
8. 轨迹完成后，将 `Label` 设置为 0；
9. 点击“停止采集”；
10. 称量本次铲取泥浆质量；
11. 在实验记录表中记录本组参数、取料质量和异常情况；
12. 清理铲具；
13. 恢复泥浆表面；
14. 进入下一组实验。

# 10. 数据后处理方法

## 10.1 数据读取

对每个实验目录读取：

```text
ur10_ft300_realtime_data.csv
```

重点使用以下字段：

```text
epoch_time
Time
Act_q0~Act_q5
Act_qd0~Act_qd5
Act_I0~Act_I5
Act_X, Act_Y, Act_Z
Fx,Fy,Fz

Label
ur_fresh

```

建议只对 `Label = 1` 的区间进行指标计算，因为这个区间对应真实铲泥动作。

---

## 10.2 零点修正

实验开始后，在铲具未接触泥浆前取一段空载数据，计算 FT300 零点：

[
F\_{x0},F\_{y0},F\_{z0},M\_{x0},M\_{y0},M\_{z0}
]

修正公式为：

[
F\_x'=F\_x-F\_{x0}
]

[
F\_y'=F\_y-F\_{y0}
]

[
F\_z'=F\_z-F\_{z0}
]

力矩同理。

---

## 10.3 指标计算

对每次实验计算：


| 指标               | 计算来源                 |
| ------------------ | ------------------------ |
| (m)                | 电子秤                   |
| (F\_{\\max})       | FT300                    |
| (F\_{\\text{avg}}) | FT300                    |
| (M\_{\\max})       | FT300                    |
| (I\_{\\text{avg}}) | UR 关节电流              |
| (E\_F)             | FT300 合力 + UR TCP 位移 |
| (J\_F)             | FT300 合力变化率         |
| (N\_s)             | FT300 力突变次数         |
| (T)                | `epoch_time`或 UR`Time`  |

# 11. 多目标综合评价

本实验不是单纯追求铲泥量最大，而是要兼顾：

```text
取料质量高
受力冲击小
能耗低
轨迹平稳
机械臂负载合理
```

因此采用归一化综合评分。

## 11.1 正向指标

取料质量 (m) 越大越好：

[
m'=\\frac{m-m\_{\\min}}{m\_{\\max}-m\_{\\min}}
]

---

## 11.2 负向指标

受力、能耗、力突变等越小越好：

[
F'=\\frac{F\_{\\max,\\max}-F\_{\\max}}{F\_{\\max,\\max}-F\_{\\max,\\min}}
]

[
E'=\\frac{E\_{\\max}-E}{E\_{\\max}-E\_{\\min}}
]

[
S'=\\frac{J\_{F,\\max}-J\_F}{J\_{F,\\max}-J\_{F,\\min}}
]

[
I'=\\frac{I\_{\\max}-I\_{\\text{avg}}}{I\_{\\max}-I\_{\\min}}
]

---

## 11.3 综合评分

建议初始权重设为：

[
Q=0.35m'+0.20F'+0.20E'+0.15S'+0.10I'
]

其中：


| 指标 | 权重 | 原因                     |
| ---- | ---: | ------------------------ |
| (m') | 0.35 | 取料效果是主要目标       |
| (F') | 0.20 | 受力过大会导致卡滞或风险 |
| (E') | 0.20 | 反映作业效率             |
| (S') | 0.15 | 反映轨迹平稳性           |
| (I') | 0.10 | 反映机械臂关节负载       |

最终选择 (Q) 最大的参数组合作为较优铲泥参数。

# 13. 安全控制要求

UR10 用户手册中强调，机器人应用需要进行风险评估，集成商需要确保完整系统不存在重大危险，并应建立适当安全措施和安全配置。

结合你的铲泥实验，建议设置以下安全措施：

## 13.1 机械安全

1. 泥浆桶必须固定，避免被铲具推动；
2. 铲具边缘不得朝向人员；
3. 实验人员不得站在机械臂运动范围内；
4. 示教前先进行空载运行；
5. 末端夹持器和 FT300 螺栓必须拧紧；
6. 急停按钮必须随时可触达。

---

## 13.2 参数安全

建议初始限制：


| 参数      | 安全建议            |
| --------- | ------------------- |
| (d)       | 初次不超过 20 mm    |
| (\\theta) | 初次不超过 30°     |
| (v\_c)    | 初次不超过 0.05 m/s |
| (v\_l)    | 初次不超过 0.05 m/s |

若预实验稳定，再逐步提高到最大水平。

---

## 13.3 力阈值安全

建议设定人工或程序阈值：


| 指标                   |           建议阈值 |
| ---------------------- | -----------------: |
| (F\_{\\max})           |          80～100 N |
| (M\_{\\max})           |           8～10 Nm |
| (N\_s)                 | 明显突变频繁时停止 |
| Robot/Safety Mode 异常 |           立即停止 |

一旦出现：

```text
铲具卡死
机器人报警
FT300 力突增
泥浆桶移动
夹具松动
```

必须立即停止实验。

# 15. 改进后的实验方案总结
