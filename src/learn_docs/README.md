# ROS 2 学习笔记索引

这个目录存放通用的 ROS 2 学习笔记，已按主题拆分为更容易查找的子目录。

## 分类目录

- [Python开发](./Python开发/README.md)
  - Python 包结构、`setup.py`、节点入口、launch 与开发流程。
- [C++开发](./C++开发/README.md)
  - 编译、构建与工作空间安装结果相关的基础知识。
- [机器人描述与RViz](./机器人描述与RViz/README.md)
  - 工作空间目录结构、包资源组织，以及与模型资源加载相关的基础认知。

## 推荐阅读顺序

1. 先看 `机器人描述与RViz/ROS文件夹结构.md`
2. 再看 `C++开发/编译基础.md`
3. 然后看 `Python开发/依赖以及setup_py.md`
4. 再看 `Python开发/核心文件写法.md`
5. 最后看 `Python开发/编写一个功能包的流程.md`

## 使用建议

- 先看每篇前面的“一句话理解”
- 真正动手前，再看后面的命令示例和排错部分
- 如果后续有新的实践经验，优先补充到现有文档中，不要重复新增同主题文件
