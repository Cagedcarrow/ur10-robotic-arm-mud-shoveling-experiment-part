# 项目文档索引

这个目录存放和当前工程直接相关的说明文档，重点记录已经落地的修改、启动方式和维护约定。

## 分类目录

- [工作区功能包分析](./工作区功能包分析/README.md)
  - 记录 `ur10_ws/src` 下全部 ROS 2 功能包的职责、入口、包间关系和当前推荐使用方式，适合忘记包用途时快速回忆。
- [Gazebo 显示与启动修复备忘](./gazebo_显示与启动修复备忘/README.md)
  - 记录 `my_robot` 在 Gazebo 中显示异常、启动缓慢、控制器不起来、残留进程干扰等问题的完整排查与修复过程，面向 ROS 2 新手。
- [机器人描述与RViz](./机器人描述与RViz/README.md)
  - 记录 URDF/Xacro、mesh 路径、RViz2 启动方式、坐标系调整等和模型显示直接相关的文档。

## 推荐阅读顺序

1. 如果你忘了某个功能包到底是干什么的，先看 `工作区功能包分析/README.md`
2. 如果你当前关心 Gazebo 启动慢、模型不显示、控制器不起来，再看 `gazebo_显示与启动修复备忘/README.md`
3. 如果你想看模型组织、RViz2 显示和末端坐标系整理，再看 `机器人描述与RViz/README.md`
4. 再阅读具体的项目改造记录
5. 如需补基础知识，再去 `learn_docs/` 查看通用笔记

## 维护约定

- 与当前工程直接相关的“改造记录”和“使用说明”放在 `docs/`
- 通用学习笔记和 ROS 2 基础整理放在 `learn_docs/`
- 如果某个文档已经失效，优先修改原文档，不要新增重复说明文件
