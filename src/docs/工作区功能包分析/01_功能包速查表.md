# 功能包速查表

## 这篇文档解决什么问题

如果你只是想快速回忆：

- 包名是什么
- 这个包一句话是干什么的
- 它是不是当前主线
- 我大概应该从哪个文件开始看

那这篇就是最快入口。

## 速查表

| 包名 | 主要职责 | 典型入口 | 主要依赖关系 | 当前状态 |
| --- | --- | --- | --- | --- |
| `my_robot` | 当前龙门架整机模型、Gazebo 启动、控制器配置主包 | `launch/sim_planning_gantry.launch.py` | 被 `my_robot_moveit_config` 使用，也会调用 `ur10_examples_py`、`ur10_perception` | 当前主线 |
| `my_robot_moveit_config` | `my_robot` 的 MoveIt 规划与 RViz 配置包 | `launch/my_robot_moveit.launch.py` | 依赖 `my_robot` 提供模型描述 | 当前主线 |
| `ur10_description` | 原始 UR10 + gantry 描述封装层 | `urdf/ur10_sim.urdf.xacro` | 被 `ur10_moveit_config`、`ur10_simulation_bringup` 使用 | 基础支撑 |
| `ur10_moveit_config` | 原始 UR10 主线的 MoveIt 配置包 | `launch/ur_moveit.launch.py` | 依赖 `ur10_description` | 基础支撑 |
| `ur10_simulation_bringup` | 原始 UR10 主线总编排包 | `launch/complete_simulation.launch.py` | 串联 `ur10_description`、`ur10_moveit_config`、`ur10_examples*`、`ur10_perception` | 基础支撑 |
| `ur10_perception` | overhead camera、点云采集、PCD 导入、world 资源 | `launch/perception_only.launch.py` | 被原始主线和 `my_robot` 主线共同使用 | 基础支撑 |
| `ur10_examples` | C++ MoveGroupInterface 示例包 | `src/move_group_interface_demo.cpp` | 依赖 MoveIt 服务链路 | 基础支撑 |
| `ur10_examples_py` | Python 示例与实用工具包 | `moveit_py_demo.py`、`gantry_control.py` | 被原始主线和 `my_robot` 主线共同使用 | 当前主线 |
| `ur10_with_shovel` | UR10 + 铲斗末端专项模型与 runbook 包 | `launch/sim_planning_shovel.launch.py` | 建立在原始 UR10 主线之上 | 历史/专项分支 |

## 如何使用这张表

### 如果你只想知道“当前整机系统入口在哪”

看：

- `my_robot`
- `my_robot_moveit_config`

### 如果你只想知道“点云和世界文件在哪”

看：

- `ur10_perception`

### 如果你只想知道“脚本工具在哪”

看：

- `ur10_examples_py`

### 如果你想知道“旧系统怎么搭起来的”

看：

- `ur10_description`
- `ur10_moveit_config`
- `ur10_simulation_bringup`

### 如果你想知道“铲斗专项版本在哪”

看：

- `ur10_with_shovel`
