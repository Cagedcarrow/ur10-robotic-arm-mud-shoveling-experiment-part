## 2. 关键指标解释（基于 README）

### 2.1 `F_max`
- **定义**：合力 `F_resultant = sqrt(Fx²+Fy²+Fz²)` 的最大值，反映实验过程中末端所受阻力的峰值强度。  
- **本实验值**：**12.166 N**  
- **解读**：峰值力约 12.2 N，可用于评估机械臂作业时是否出现瞬时高阻力。因无参考实验对照，无法判定是否异常，但结合后续冲击计数（0），未出现剧烈突发力矩。

### 2.2 `E_force`
- **定义**：末端机械作业能量估计，`E_force = sum( F_resultant * ds )`，其中 ds 为 TCP 位移微元。非电能，表示克服阻力沿轨迹累积的近似功。  
- **本实验值**：**14.684 J**  
- **解读**：较低能量消耗，可能因为全段数据包含大量静止或缓慢运动时段。该值可信度受 `F_resultant` 计算中的平滑滤波（window=5）及采样质量影响。

### 2.3 `smoothness_force`
- **定义**：`mean(|ΔF|/Δt)`，即合力变化率的均值，越小表示力输出越平稳。  
- **本实验值**：**0.723 N/s**  
- **解读**：数值较小，说明实验过程中力波动较缓慢，无频繁冲击。与 `shock_count=0` 一致。但需要注意该值是基于全段数据（包含静止段）计算，可能人为压低均值，实际作业段平滑性可能更高（或更低）。

### 2.4 `I_mean`
- **定义**：关节电流总和 `I_sum = |I0|+...+|I5|` 的均值，反映关节平均负载水平。单位隐含为 A（安培）或 mA，根据原始 `processed_metrics.csv` 数值可推测为 A。  
- **本实验值**：**8.431 A**  
- **解读**：平均关节总电流约 8.43 A，反映机械臂在实验过程中的平均电机负载。高电流可能表示阻力较大或运动速度较高，但受缺失质量影响，难以区分。由于无 `mass_g`，不能计算单位质量负载，需结合后续实验对比。

## 4. 结论

本次实验（20260430_212937）存在 **数据质量严重不足** 的问题，主要表现为：
1. **未标记实验段（Label=1 缺失）**，未使用作业段而采用全段数据，导致分析失去针对性；
2. **FT300 数据刷新率仅 4.8%**，极低的新鲜度使所有力/力矩相关指标（F_max, E_force, smoothness_force, M 系列等）可信度极低，不能反映真实铲取过程；
3. **负载质量（mass_g）缺失**，限制了能量效率分析。

在此条件下，即使部分指标如 `F_max=12.2 N`，`smoothness_force=0.72 N/s` 表面良好，实则应视为 **无效或不可靠**。该实验不适宜用于精细作业评估，仅可作为数据采集链路测试的参考。
